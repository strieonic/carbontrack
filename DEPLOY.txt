VERCEL DEPLOYMENT INSTRUCTIONS FOR CARBONTRACK
==============================================

STEP 1: Install Vercel CLI
---------------------------
Run this command in your terminal:
npm install -g vercel

Wait for installation to complete.


STEP 2: Login to Vercel
------------------------
Run:
vercel login

Choose your login method (GitHub, Email, etc.)
Follow the prompts to authenticate.


STEP 3: Link Your Project (First Time)
---------------------------------------
In your project directory (D:\carbontrack), run:
vercel link

Answer the prompts:
- Set up and deploy? Yes
- Which scope? Choose your account
- Link to existing project? Yes (if you have carbontrack-tau)
- What's the name? carbontrack-tau
- Or choose "No" to create new project


STEP 4: Deploy to Production
-----------------------------
Run:
vercel --prod

This will:
- Build your project
- Upload files to Vercel
- Deploy to production
- Give you the live URL


ALTERNATIVE: Deploy via Vercel Dashboard
-----------------------------------------
1. Go to: https://vercel.com/dashboard
2. Click "Add New" > "Project"
3. Import from GitHub: strieonic/carbontrack
4. Configure:
   - Framework Preset: Other
   - Root Directory: ./
   - Build Command: (leave empty)
   - Output Directory: (leave empty)
5. Click "Deploy"


IMPORTANT: After Deployment
----------------------------
1. Your site will be live at: https://carbontrack-tau.vercel.app
2. Test all pages work
3. Check browser console for errors
4. Remember: Supabase key still needs to be updated in js/supabase.js


TROUBLESHOOTING
---------------
If deployment fails:
1. Check vercel.json is valid
2. Ensure all files are committed to git
3. Try: vercel --prod --force
4. Check Vercel dashboard for error logs
